About the application

This web application can hugely simplify the hassle of choosing online product. This web application will try to solve issue of reading thousands of reviews by scraping all the reviews and then applying NLP to obtain summarisation of positive and negative reviews.

  • Easy to use. Just type link of product
  • Gives accurate result most of the time
  • Reasonably fast and secure

The web application was created as a part of SGP project. The backend was built using Flask frameword, Selectorlib and Microsoft Text Analytics. The frontend was made using Bootstrap.
